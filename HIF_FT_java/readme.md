
# HIF Fachtheorie Seed

A seed project to be used for
HIF Fachtheorie examinations.

## Aufgabe 1

Will be about implementing a class
model which is mapped into a relational
database.

Mind the resulting database model.

Implement tests to ensure your solution 
works.

## Aufgabe 2

Will be about implementing some business
logic calculations using mostly basic
language features.

No ft_ormapping.persistence and no frontend needs to be
implemented.

Generate the required test data inside
your tests.


## Aufgabe 3

A RESTful API implementation. Show case
your skills in implementing and testing
a RESTful API.

No ft_ormapping.persistence implementation is expected.

Generate the required test data inside your
tests.
